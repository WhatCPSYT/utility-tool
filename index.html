Utility Tool

Wybierz wersję do pobrania:

Pobierz dla Windows (.bat) Pobierz dla Linux/Lubuntu (.sh)

Instrukcja dla Windows

  1. Pobierz plik windows_tool.bat.
  2. Kliknij prawym przyciskiem myszy i wybierz Uruchom jako administrator.
  3. Postępuj zgodnie z menu w programie.

Instrukcja dla Linux/Lubuntu

  1. Pobierz plik linux_tool.sh.
  2. Otwórz terminal i przejdź do folderu, gdzie zapisano plik:
    • Jeśli plik jest na Pulpicie (polska wersja): 
      cd ~/Pulpit
    • Jeśli plik jest na Desktop (angielska wersja): 
      cd ~/Desktop
    • Jeśli plik jest w Pobrane (polska wersja): 
      cd ~/Pobrane
    • Jeśli plik jest w Downloads (angielska wersja): 
      cd ~/Downloads
  3. Nadaj prawa do uruchamiania: 
    chmod +x linux_tool.sh
  4. Uruchom program jako administrator: 
    sudo ./linux_tool.sh
  5. Postępuj zgodnie z menu w programie.
